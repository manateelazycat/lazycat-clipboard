  # Lazycat Clipboard

  A modern, elegant clipboard manager for seamless text and image sharing across devices.
</div>

## Features

- **Text Sharing** - Quickly send and receive text content
- **Image Sharing** - Support for image upload, drag-and-drop, and paste
- **Drag & Drop Reordering** - Easily reorganize clipboard items
- **Keyboard Shortcuts** - Efficient navigation with keyboard support
- **Responsive Design** - Works seamlessly on desktop and mobile devices

## Tech Stack

- **Vue 3** - Progressive JavaScript framework
- **TypeScript** - Type-safe development
- **Vite** - Next-generation frontend tooling
- **Tailwind CSS** - Utility-first CSS framework
- **Dexie.js** - IndexedDB wrapper for local storage
- **vuedraggable** - Drag-and-drop functionality

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/aspect-apps/lazycat-clipboard.git

# Navigate to project directory
cd lazycat-clipboard

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `↑` / `K` | Select previous item |
| `↓` / `J` | Select next item |
| `Enter` | Copy selected item |
| `E` | Edit selected item |
| `Tab` | Focus on input box |
| `Escape` | Close modal |

## Usage Tips

- Use Lazycat NetDisk for file sharing
- Drag and drop to reorder list items
- Click or press Enter to copy
- Press ↑, ↓, J, K, E for quick navigation
