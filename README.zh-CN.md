  # 懒猫剪贴板

  一款现代、优雅的剪贴板管理器，实现跨设备无缝共享文字和图片。
</div>

## 功能特点

- **文字共享** - 快速发送和接收文字内容
- **图片共享** - 支持图片上传、拖拽和粘贴
- **拖拽排序** - 轻松重新排列剪贴板项目
- **键盘快捷键** - 高效的键盘导航支持
- **响应式设计** - 完美适配桌面端和移动端

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **TypeScript** - 类型安全的开发体验
- **Vite** - 下一代前端构建工具
- **Tailwind CSS** - 实用优先的 CSS 框架
- **Dexie.js** - IndexedDB 封装库
- **vuedraggable** - 拖拽排序功能

## 快速开始

### 环境要求

- Node.js 18+
- npm 或 yarn

### 安装

```bash
# 克隆仓库
git clone https://github.com/aspect-apps/lazycat-clipboard.git

# 进入项目目录
cd lazycat-clipboard

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 生产构建

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `↑` / `K` | 选择上一项 |
| `↓` / `J` | 选择下一项 |
| `Enter` | 复制选中项 |
| `E` | 编辑选中项 |
| `D` / `Delete` | 删除选中项 |
| `Tab` | 聚焦到输入框 |
| `Escape` | 关闭对话框 |

## 使用技巧

- 共享文件请用懒猫网盘
- 右边列表支持拖拽排序
- 点击或回车即复制
- 快捷键 ↑、↓、J、K、E
